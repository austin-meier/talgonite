# Version 0.1.7 (2026-05-05)

## Message formatting

Messages and profiles now correctly show any colours inside of the color blocks (`{=bred` shows 'red' in red colour). As a bonus _italic_ text (use single `*` or `_`) and **bold** (use double `*` or `__`) formatting has been added.

## Group support (thanks @Jinori)

Full group support added! Groups can be requested, requests can be accepted/declined, members can be kicked, etc etc.

## More action bar hotkeys (and emotes)

More rows of hotkeys can be assigned to the action bar. By default the action bar will now have 3 rows of emotes in these slots with the keys bound to Ctrl, Alt and Ctrl+Alt.

Basically this adds the emote hotkeys from the official client, but those slots in the action bar can be set to something else instead if you'd rather.

## Social status changing (thanks @wren11)

You can change your status to 'Lone Hunter', 'Need Help' and the rest through a dropdown on your profile.
